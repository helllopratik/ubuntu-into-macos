═════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═════════════════════════════════════════════════════════════════

IF INSTALLATION FAILS:
1. Check internet connection
2. Ensure ~500 MB free space
3. Verify sudo access
4. Try without backup option
5. Check system requirements

IF THEME DOESN'T SHOW:
1. Reboot (essential!)
2. Verify via GNOME Tweaks
3. Check ~/.themes directory
4. Try manual gsettings
5. Restart GNOME Shell

IF SOUNDS DON'T WORK:
1. Check volume settings
2. Verify sound files installed
3. Test with paplay command
4. Check PulseAudio status
5. Review troubleshooting guide

IF SCRIPTS FAIL:
1. Check WhiteSur compatibility
2. Review error messages
3. Try manual installation
4. Update to latest version
5. Check troubleshooting guide
